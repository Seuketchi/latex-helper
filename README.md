## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+C` | Insert Citation |
| `Ctrl+Shift+F` | Insert Figure |
| `Ctrl+Shift+T` | Insert Table |

## Usage

1. Open a LaTeX project in VS Code
2. Click the book icon in the Activity Bar to open LaTeX Helper
3. Use the sidebar panels to navigate and insert content

## Requirements

- VS Code 1.85.0 or higher
- LaTeX files (.tex) in your workspace
- Optional: .bib files for bibliography features

## Installation

Install from VSIX:
1. Download the .vsix file
2. In VS Code, go to Extensions > ... > Install from VSIX
3. Select the downloaded file
